# Monty Hall

Jogo de porta premiada (Monty Hall), a qual o usuário pode escolher a quantidade de portas e a porta a qual está premiada.

## Início

### Pré-requisitos:

Você precisa do NodeJS e NPM instalados em sua máquina.

```
npm install
npm run dev
```

## Construído com:

for Production.
